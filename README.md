# 📰 Redesign do Site — Agenda Maranhão  
Espaço de Produção Jornalística Coletiva  
`www.agendamaranhao.com.br` [(agendamaranhao.com.br in Bing)](https://www.bing.com/search?q="http%3A%2F%2Fwww.agendamaranhao.com.br%2F")

Este repositório documenta o projeto de **redesign do site Agenda Maranhão**, um portal dedicado à produção jornalística independente, com foco em **fotografia, memória, turismo, arte, ciência, tecnologia e inovação**.  
O objetivo do redesign é modernizar a experiência do usuário, fortalecer a identidade editorial e tornar o conteúdo mais acessível e visualmente atraente.

---

## ✨ Sobre o Agenda Maranhão  
O Agenda Maranhão é um espaço colaborativo de jornalismo cultural e científico, reunindo reportagens, ensaios fotográficos, entrevistas e análises sobre temas relevantes para o estado do Maranhão e para o Brasil.

### **Administradores e Fundadores**
- **Ed Wilson Araújo** — Jornalista, professor da Universidade Federal do Maranhão (UFMA) e doutor pela Pontifícia Universidade Católica do Rio Grande do Sul (PUC-RS).  
- **José Reinaldo Martins** — Jornalista e mestre pela Universidade de São Paulo (USP).

---

## 🎯 Objetivo do Redesign  
O redesign foi planejado para:

### **1. Modernizar a identidade visual**  
- Layout mais limpo, leve e contemporâneo  
- Hierarquia visual aprimorada  
- Maior foco em fotografia e conteúdo editorial  

### **2. Melhorar a experiência do usuário (UX)**  
- Navegação simplificada  
- Conteúdo mais organizado  
- Melhor leitura em dispositivos móveis  

### **3. Reforçar o caráter cultural e jornalístico do portal**  
- Destaque para reportagens e ensaios fotográficos  
- Espaço mais claro para temas como memória, turismo e inovação

## 🚀 Como Executar o Projeto (Ambiente Local)  
1. Clone o repositório:  
   ```bash
   git clone https://github.com/seu-usuario/Redesign_Agenda_Maranhao_Website.git
   ```
2. Abra o diretório no VS Code  
3. Caso esteja usando WordPress localmente, configure o ambiente (XAMPP, LocalWP, Laragon etc.)  
4. Importe ou copie os arquivos do tema para a pasta `wp-content/themes`  
5. Ative o tema no painel WordPress
